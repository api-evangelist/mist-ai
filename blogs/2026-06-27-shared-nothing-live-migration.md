Hi I know that today it is not possible to perform shared-nothing live migration of VMs - that is, live migration of VMs between hosts without shared storage involved. This is possible to achieve manually using virsh commands, however I guess not officially supported by HPE and also very "manual" work with some cleanup etc. involved.
